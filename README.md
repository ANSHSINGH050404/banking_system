# banking_system

Banking system demo with accounts, transfers, and core banking flows.

## Stack

TypeScript

## Getting started

```bash
git clone https://github.com/ANSHSINGH050404/banking_system.git
cd banking_system
bun install
bun run dev   # or: bun run start
```
